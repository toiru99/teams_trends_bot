# Lab Context — AI Trends Filter 보조 입력

이 파일은 `ai-trends-filter` 스킬이 매 cron 실행마다 자동으로 로드한다.
비어 있어도 동작하지만(5축 일반 분류만 적용), 채워넣을수록 Step 2 (진행 과제 매핑) 점수가
정확해져서 게재 정확도가 올라간다.

> 이 파일은 자유 양식이지만 아래 4개 섹션을 채우는 것이 권장된다.
> 빈 섹션은 그대로 두거나 삭제해도 무방.

---

## 1. 현재 진행 중 과제

- (예시) 제안서 — 2026 정통부 공모 "직무별 AI 에이전트 템플릿" (마감 2026-07-15)
- (예시) 사내 도입 트랙 — XX팀 RAG 어시스턴트 베타 (8월 GA 목표)
- (예시) 대회 — NeurIPS 2026 LLM Efficiency Track 출전 검토 중

## 2. 사내 도구·시스템

- (예시) `agent-gateway` — 운영 중 멀티 LLM 라우팅 게이트웨이. OpenAI 호환 API.
- (예시) `harness-cli` — 사내 평가/회귀 도구. pytest + ragas + 사내 데이터셋.
- (예시) `voice-pipeline` — STT→LLM→TTS 통합 파이프라인 (현재 Whisper + Claude + edge-tts).

## 3. 시한이 있는 의사결정

- (예시) 2026-06-30 — 신규 코드 어시스턴트 모델 비교 보고서 마감 (현재 Claude/GPT/Kimi 후보)
- (예시) 2026-07-15 — 위 공모 제안서 제출

## 4. 회피·제외 키워드 (선택)

- (예시) "AI 윤리 일반론" 류 기사 — 응용 통찰 없으면 폐기
- (예시) 펀딩/M&A 뉴스 — 컨텍스트 매핑 없으면 폐기
